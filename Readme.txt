1.在main.h里定义#define Referee_System 即开启裁判系统限制底盘功率、根据等级调整射速射频

2.当未定义裁判系统时，可修改在ShootControl.c文件头的int robotLevel=1;手动调整射击模式

3.射速调整修改ShootControl.c里的Shoot_UP=130;原定义130对应射速约为25-27m/s，偶有超过30.注意修改时同时调整等级相关的射速设置。

存在问题：
1.当前无法读取枪口热量，原因不明

2.在上电烧程序重启后，较经常开机后就进入出错模式，原因不明。点击stm32上的重启按键后或断电重启后则问题解决。
正常开机下基本不出现此情况，若出现请留意出现原因。

3.在工控时刻发送数据时，单片机在串口2初始化时会出错，要确保单片机比工控开机早或在调试时候断开单片机串口2rx
